## 🚀 Featured Project

### [MinhaVenda](https://github.com/rodrigosantoscosta/minhavenda-nestjs)
> Full-stack e-commerce platform built from scratch with a clean architecture mindset.

| Layer | Technology | Highlights |
|---|---|---|
| **API** | NestJS + TypeScript | Domain-driven design, use-case layer, RabbitMQ events |
| **Database** | PostgreSQL + TypeORM | Full-text search (`tsvector`), cursor pagination |
| **Frontend** | React + Vite | Admin dashboard, DRE module, dynamic filters |
| **Testing** | Jest + Playwright | Unit, integration (Testcontainers) & E2E coverage |

**Key implementations:**
- 🔎 Full-text search with `tsvector` (Portuguese dictionary) + ranked results
- 📄 Cursor-based pagination with opaque base64 tokens for infinite scroll
- 🧪 Integration tests using **Testcontainers** with real PostgreSQL instances
- 🎭 End-to-end test suite covering critical user flows with **Playwright**
- 📊 Admin financial dashboard (DRE) with date-range filtering
